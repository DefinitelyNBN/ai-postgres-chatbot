# ai-postgres-chatbot
AI-powered chatbot that interacts with PostgreSQL databases using natural language queries. Built with LangChain and supports automatic SQL generation from natural language input.
## Cloning and Running Locally

Follow these steps to clone and run this project on your local computer:

### 1. Prerequisites

- **Git**: Make sure Git is installed on your system.
- **Node.js and npm**: Install Node.js (which includes npm).
- **PostgreSQL**: Ensure PostgreSQL is installed and running.
- *(Optional)* **Python**: If the project uses Python, install Python and pip.

### 2. Clone the Repository

```bash
git clone https://github.com/DefinitelyNBN/ai-postgres-chatbot.git
cd ai-postgres-chatbot
